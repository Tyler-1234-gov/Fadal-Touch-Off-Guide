Fadal CNC - Tool Touch-Off Procedure

Step 1: Prepare the Machine
  • Ensure the tool magazine is loaded with the correct tools.
  • Place a precision tool setter or a gauge block on the machine table.
Step 2: Go to Tool Setting Cycle
  • Press MANUAL until the command prompt page appears.
  • Type UT and press ENTER to get to the utilities page.
  • Select option number 1 - Tool Setting Cycle, then press ENTER.
  • Select the starting and ending tool number. If only setting one tool, start and end with that tool number.
  • Select Jog to Position. DO NOT press ENTER.
  • The prompt will ask for gauge block size. Enter the size in inches and press ENTER.
Step 3: Move the Tool to the Touch-Off Position
  • Select option number 1 - Enter Tool Diameter, then press ENTER.
  • Input the tool diameter and press ENTER.
  • Select option number 2 - Jog to Position, then press ENTER.
  • Press JOG to enable jogging mode.
  • Use the axis and increment knobs along with the handwheel to move the tool slowly towards the tool setter or gauge block.
  • If using a tool setter gauge, lower the tool until it reads zero.
  • If using a feeler gauge, move the tool down until light contact is made.
  • Press MANUAL to abort.
  • Press CYCLE START to return the head to Z zero.
  • The tool changer will automatically call the next tool if touching off consecutive tools. Repeat Step 3 until all tools are measured.
Step 4: Move to Machine Home
  • Press MANUAL until the command prompt page appears.
  • Input HO and press ENTER.
Step 5: Verify Tool Offsets
  • Input DT from the command prompt page and press ENTER.
  • Verify tool offsets have been input.
